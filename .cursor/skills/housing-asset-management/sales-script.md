# 话术模板与应对策略

## 话术生成规则

生成话术时遵循以下原则：
- **口语化**：像朋友聊天，不要书面腔
- **数据驱动**：每个论点都有报告中的数据支撑
- **节奏感**：先建立信任 → 再讲模式 → 最后促成
- **与报告同步**：话术顺序对应报告章节，方便边翻报告边说

## 话术结构模板

### 开场（对应报告封面）

```
[业主称呼]您好，我是省心租的[自己名字]。
今天专门为您准备了一份房屋托管方案，
您看一下，这是根据您[地址]这套房子定制的。
```

**要点**：强调「定制」，让业主感受到重视。

### 模式介绍（对应报告-服务模式）

```
我们省心租跟传统中介不一样，我们不赚差价。
您的房子租多少钱，就是多少钱，完全透明。
我们只收一个服务费，就是每个月3天的租金，
相当于一个月30天，您拿27天，我们拿3天。
```

**要点**：先讲「不赚差价」建立信任，再讲服务费。
3天租金比说「10%」更容易接受。

### 费用讲解（对应报告-费用明细）

```
您看这个表，您这套房子我们建议月租[X]元，
每个月服务费就是[Y]元，您实际到手[Z]元。
一年下来您的收入是[年收入]元。

而且我跟您说，我们签约的时候，
会先给您[保证金金额]元的保证金，
这笔钱签约就给您，不用等租出去。
```

**要点**：先讲收入，再讲保证金，制造惊喜感。

### 服务承诺（对应报告-服务承诺）

```
我们承诺30天之内帮您租出去。
在这之前，我们会免费帮您做一次深度保洁，
如果有需要维修的地方，我们也免费帮您修。
还会给您装一个智能门锁，这些都不要钱。

万一30天没租出去呢？
那保证金归您，保洁维修的钱我们也不要了，
您一分钱不用出。所以对您来说是零风险的。
```

**要点**：这是最有说服力的部分，要讲清楚「零风险」。

### 促成（对应报告-合作流程）

```
整个流程很简单，您看这个时间线：
签约之后我们马上安排保洁维修，
然后上架推广，筛选租客，
后面的事情您都不用操心了。

您觉得这个方案怎么样？
```

## 常见疑虑应对

### "服务费太贵了"
```
您想一下，3天的租金，我们帮您搞定所有事情——
找租客、保洁、维修、门锁、日常管理，
您完全不用操心。
自己租的话，光空置一个月就亏了整月租金，
而且还要自己处理各种事情。
3天换27天的省心，其实很划算的。
```

### "我自己也能租出去"
```
确实，您自己也能租。
但您想一下，自己租要花时间带看、筛选租客，
万一遇到不好的租客，后面更麻烦。
而且空置期的损失远比服务费高。
我们专业团队30天保证租出去，
租不出去保证金还归您，您没有任何损失。
```

### "保证金会不会有问题"
```
保证金是签约就给您的，白纸黑字写在合同里。
这笔钱就是我们的诚意，
说明我们对自己的服务有信心。
30天租不出去，这钱就是您的了。
```

### "智能门锁安全吗"
```
这个门锁是我们统一采购的品牌锁，
主要是方便管理，每个租客有独立密码，
退租自动失效，比传统钥匙安全多了。
而且这个锁是免费给您装的，合同到期锁也是您的。
```
