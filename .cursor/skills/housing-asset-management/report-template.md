# HTML 报告模板规范

## 设计规范

### 风格：专业商务风
- **主色调**：深蓝 `#1a2332`，搭配金色点缀 `#c9a84c`
- **背景色**：`#f5f6fa`（浅灰）
- **卡片背景**：`#ffffff`
- **正文字色**：`#2c3e50`
- **辅助字色**：`#7f8c8d`
- **成功/安全色**：`#27ae60`
- **字体**：`"PingFang SC", "Microsoft YaHei", sans-serif`

### 布局要求
- 最大宽度 900px，居中
- 卡片式布局，圆角 12px，轻微阴影
- 各章节间距 24px
- 响应式：移动端 padding 缩小，字号适配

### 打印适配
- 包含 `@media print` 样式
- 隐藏非必要装饰元素
- 确保黑白打印可读

## 报告结构

HTML 报告必须包含以下章节，按顺序排列：

### 1. 报告封面/头部
```html
<!-- 公司 Logo + 名称 -->
<!-- 报告标题：「房屋托管服务方案」 -->
<!-- 副标题：专属于 [业主姓名] 的定制方案 -->
<!-- 生成日期 -->
```

### 2. 房屋信息卡片
```
展示内容：
- 业主姓名
- 房屋地址
- 户型 / 面积
- 楼层（如有）
- 装修情况（如有）
- 配套设施（如有）
- 周边配套（如有）
```

使用图标或标签式布局，信息一目了然。

### 3. 服务模式说明
核心信息：
- 「不赚差价，只收服务费」的模式说明
- 服务费计算方式：每月仅收取 3 天租金
- 对比传统中介模式的优势

用简洁的图示或对比表格展示。

### 4. 费用明细
必须包含一个清晰的费用表格：

```
| 项目           | 金额      |
|----------------|-----------|
| 建议月租金     | ¥X,XXX   |
| 每月服务费     | ¥XXX     |
| 您的月净收入   | ¥X,XXX   |
| 保证金(签约即付)| ¥X,XXX  |
| 预估年净收入   | ¥XX,XXX  |
```

**关键**：用大字号高亮「您的月净收入」和「保证金」。

### 5. 服务承诺
用卡片/图标列表展示：
- 30天出租承诺（未出租则保证金+维修费归业主）
- 免费专业保洁
- 免费基础维修
- 免费智能门锁安装
- 专业租客筛选
- 租期内全程管理

### 6. 合作流程时间线
用时间线/步骤条展示：
1. 签约 → 支付保证金给业主
2. 房屋准备 → 保洁、维修、安装门锁
3. 上架推广 → 多渠道发布房源
4. 租客入住 → 严格筛选后入住
5. 持续管理 → 日常维护、租金代收

### 7. 联系方式/页脚
```
省心租
联系电话：138-8888-6688
「让房东省心，让租客安心」
```

## 技术要求

- 单文件 HTML，内联所有 CSS 和 JS
- 使用 CSS 变量管理主题色
- 不依赖任何外部 CDN 或资源
- 使用语义化 HTML 标签
- 数据通过 JS 变量注入，方便后续系统对接

```javascript
// 报告数据结构（放在 <script> 中）
const reportData = {
  owner: {
    name: "张三",
    phone: ""  // 可选
  },
  property: {
    address: "杭州市西湖区XX路XX号XX室",
    layout: "2室1厅1卫",
    area: 89,
    floor: "12/18",        // 可选
    decoration: "精装修",   // 可选
    facilities: [],         // 可选
    surroundings: []        // 可选
  },
  pricing: {
    monthlyRent: 4500,
    serviceFee: 450,        // 自动计算: monthlyRent / 30 * 3
    netIncome: 4050,        // 自动计算: monthlyRent - serviceFee
    deposit: 2250,          // 自动计算: monthlyRent * 0.5
    yearlyIncome: 48600     // 自动计算: netIncome * 12
  },
  propertyStatus: {
    needsCleaning: true,
    needsRepair: false,
    repairDetails: ""
  },
  generatedDate: "2026-03-05"
};
```

所有金额计算由 JS 自动完成，只需传入 `monthlyRent` 即可。
